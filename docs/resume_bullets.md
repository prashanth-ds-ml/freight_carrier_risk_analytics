# Resume Bullets

- Built a carrier risk analytics workflow using FMCSA company census, inspection, and crash data to identify carrier identity gaps, authority status issues, inspection risk, crash history, and data quality exceptions.
- Designed SQL validation checks for carrier identity completeness, inspection out-of-service rates, crash-history signals, and unmatched records across heterogeneous FMCSA datasets.
- Planned a Power BI risk dashboard with carrier exception tracking, risk-tier segmentation, data quality monitoring, and weekly review outputs for operational decision support.
